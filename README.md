# Ecxel-Basics
📊 Ejercicios, plantillas y apuntes del curso de Excel Básico en Platzi. Incluye tablas de práctica, fórmulas y formatos aplicados.
## 🗂️ Data Organization & Table Structure

Practice file focused on clean data structure principles.

**Skills applied:**
- Correct table structure (headers, continuous data, no merged cells)
- Excel table format (Ctrl + T)
- Data validation with dropdown lists
- Conditional formatting for visual analysis
- Data slicers

**Tools:** Excel 365 — Tables, Data Validation, Conditional Formatting

## 📊 NexCore Sales Analysis 2024

Sales dataset analysis exercise simulating a real work task.

**Skills applied:**
- Data cleaning (missing values, duplicates)
- Pivot table: total revenue by region
- Pivot table: top 3 products by revenue
- Dynamic bar chart
- Business insight extraction

**Tools:** Excel 365 — Pivot Tables, Dynamic Charts, Conditional Formatting
